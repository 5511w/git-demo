1. IP地址

内网IP：家/虚拟机/校园网内部用，能局域网互通，别人外网访问不到

公网IP：全网唯一，互联网可以直接访问

查看IP
Windows：ipconfig
Linux：ip addr

2. 端口

IP定位到一台电脑，端口定位到电脑上某一服务

范围：0~65535

常用端口：80 → HTTP

443 → HTTPS

3306 → MySQL

22 → Linux 远程连接

3. HTTP/HTTPS

HTTP:明文传输，不安全，默认 80 端口

HTTPS：加密传输，安全，默认 443 端口

4. 局域网、网关、DNS

局域网：同一个路由器下的所有设备（手机、电脑、虚拟机）

网关：局域网出去上网的 “大门”

DNS：把域名（www.xxx.com）翻译成 IP 地址
